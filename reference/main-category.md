# 메인페이지의 판매차량을 분리

## 현재 상황
메인페이지에 SaleCarSection 이 존재. 판매차량으로 국산차와 수입차를 모두 보여줌

## 변경 후
### 메인페이시 섹션 분리
- 판매 차량 섹션이 장기렌트(국산차) 섹션과 리스(수입차) 섹션으로 분리됨
- 장기렌트 섹션은 국산차 DOMESTIC인 제조사의 차량들을 가져와서 보여줌
- 리스 섹션은 수입차 IMPORT인 제조사의 차량들을 가져와서 보여줌
- 두 섹션의 UI는 기본적으로 종래의 판매차량 섹션과 동일하게
- 다만 이제 '전체/국산차/수입차' 선택 버튼은 더이상 필요없음

### 헤더 수정
- 헤더의 2월 즉시 출고는 클릭했을 때 아무일도 일어나지 않게 처리
